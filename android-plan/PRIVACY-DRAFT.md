# Borrador de Politica de Privacidad

## Datos que procesa

Preven-IA procesa el texto, enlace o contenido que el usuario pega manualmente o comparte con la app para realizar una revision preventiva de senales de riesgo.

## Datos que NO procesa

Preven-IA no solicita ni necesita:

- Contrasenas.
- PIN.
- Codigos SMS.
- CVV.
- Frases semilla.
- Datos bancarios completos.
- Acceso a mensajes privados.
- Contactos.
- Ubicacion.
- Camara o microfono.

## Analisis local e IA online

El texto puede analizarse localmente en el dispositivo. Si el usuario usa una funcion de IA online, el texto enviado puede transmitirse al servicio de analisis configurado para generar una recomendacion orientativa.

## Uso de datos

No vendemos datos personales. No usamos el contenido para publicidad personalizada.

## Recomendacion al usuario

No ingreses claves, PIN, codigos SMS, CVV ni frases semilla en Preven-IA.
