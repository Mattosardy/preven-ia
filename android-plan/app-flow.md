# Flujo de App

1. Usuario abre Preven-IA.
2. Puede pegar un link o mensaje manualmente.
3. Puede recibir texto desde el menu Compartir de Android.
4. La app analiza el contenido.
5. Resultado:
   - Verde: no se detectaron senales fuertes.
   - Amarillo: revisar antes de abrir.
   - Rojo: no abrir sin revision.
6. Botones:
   - Abrir enlace.
   - Copiar informe.
   - Pedir revision humana.
   - Cancelar.

El boton Abrir enlace debe aparecer despues del resultado y con texto preventivo claro.
